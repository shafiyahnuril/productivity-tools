## MODIFIED Requirements

### Requirement: Categorization tags

The system SHALL allow notes to be tagged, and each tag color SHALL map to the exact PAPER.OS Sticky Notes theme rules for varying light and dark modes (Yellow, Pink, Blue, Green, Purple, Orange) to simulate physical paper notes.

#### Scenario: User assigns a tag to a note

- **WHEN** user selects a category tag when editing a note
- **THEN** the note is displayed adopting the specific Light (BG/Border) or Dark (BG/Border) values matching its color category according to newDesignSystem.md
