## MODIFIED Requirements

### Requirement: Theming

The system SHALL support Dark ("Muted Charcoal") and Light ("Warm Paper") themes per the `newDesignSystem.md` PAPER.OS guidelines. The UI SHALL utilize precise typography, warm off-whites or charcoals, and generous border radii without heavy drop shadows.

#### Scenario: User interacts with layout

- **WHEN** user toggles between dark to light
- **THEN** the app's components, typography, and backgrounds switch to the light mode palette automatically using PAPER.OS semantic tokens
