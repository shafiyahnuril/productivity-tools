## MODIFIED Requirements

### Requirement: Calendar view

The system SHALL display a calendar with days of the week, allowing users to view scheduled events. On mobile devices, the event list SHALL support swipe-to-action gestures.

#### Scenario: User checks today's schedule

- **WHEN** user navigates to the Calendar module
- **THEN** they see an agenda showing events for the current day

#### Scenario: User swipes event on mobile

- **WHEN** user horizontally swipes an event item to the left on a small screen
- **THEN** quick action buttons are revealed
