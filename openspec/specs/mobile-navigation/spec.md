## Requirements
### Requirement: Bottom Navigation

The system SHALL render a floating bottom navigation bar on mobile and tablet devices with shortcuts to Dashboard, Notes, Focus, Calendar, and Profile.

#### Scenario: User navigates via bottom navigation

- **WHEN** user taps a menu icon on the bottom navigation
- **THEN** the application transitions to the selected page securely with a sliding animation and the active item gets highlighted in Primary Purple
